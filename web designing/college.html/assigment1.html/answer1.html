web designing - Course curriculum

The web designing Course provides the fundamental concepts of creating and designing intereactive and responses web pages. students learn the use of

  • html
  • css
  • javascript
for front-end web development. this course helps learners build strong practical skills for devloping real-world web interfaces.

the syllabus is designed as per the guidelines of AKTU and aims to prepare students for industry-level projects.the course includes topics from basic HTML to advanced web technologies.

course modules

  1. introduction to web designing
  2. HTML basics: structure, elements & attributes
  3. text formatting and lists in HTML
  4. links, images and mutimedia
  5. tables and forms
  6. introduction to javascript
  7. web development project

key terminologies

HTML
hyper text markup language - used to design the structure of web pages.
CSS
cascading style sheets - used to design and beautify the webpage layout.
JavaScript
a programming language for web development which make web pages dynamic and intereactive.
Responsive Design
designing web pages that adapt to different screen sizes

course structure (nested list)

  1. semester 1
    • introduction to html
    • formatting tag
    • lists and hyperlinks
  2. semester 2
    • introduction to CSS
    • layout design
    • mini project

Evaluation Scheme (Mixed List)

  1. Theory
    • Mid-Semester Exam - 15 Marks
    • End-Semester Exam - 40 Marks
  2. Practical
    • Experiment Performance - 15 Marks
    • Viva Voce - 10 Marks
    • Record Submission - 15 Marks

Conclusion

By completing this course, students will be able to design creative, user-friendly, and functional web pages using HTML and CSS. They will gain confidence to pursue advanced topics like Bootstrap and JavaScript frameworks in the next semesters.
